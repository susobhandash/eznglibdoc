Properties:
Name Values Mandatory / Default Description
progress number: '%age value' Mandatory Display the progress in percentage.
radius number: 'pixel value' Default: 50px Determine the size of the circle.
progressColor string: 'hex or rgb/a value' Default: #0366d6 Determine the color of the progress bar.
progressBgColor string: 'hex or rgb/a value' Default: #cccccc Determine the color of the background of progress bar.
pathStrokeWdt number: 'pixel value' Default: 3px Determine the width of the progress bar.
progressStrokeWdt number: 'pixel value' Default: 5px Determine the width of the background of progress bar.
showText boolean: 'true/false' Default: true Determine whether to show inner circle and text inside it.
innerCircleRad number: 'pixel value' Default: (radius - 5) px Determine the size of the inner circle.
fontSize number: 'pixel value' Default: 20px Determine the size of inner text.
textCircleColor string: 'hex or rgb/a value' Default: #0366d6 Determine the color of the text circle.
textColor string: 'hex or rgb/a value' Default: #ffffff Determine the color of the text inside inner circle.
text string: 'hex or rgb/a value' Default: %age value Determine the content of the text circle. Default is the progress %age, use this to override text content.
Events:
Name Parameter Description
clickOnSvg $event Returns the current progress %age (event.progress) as well as text value (event.text).
