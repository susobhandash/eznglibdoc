Syntax Code Pattern 
        
        <ez-tabbar styleClass="custom_style_tabbar" headerStyleClass="custom_style_tabheader" bodyStyleClass="custom_style_tabbody" direction="horizontal" scrollHt=150 height=120>
          <ez-tab tabTitle="Person">
              Content
          </ez-tab>
          <ez-tab [tabTitle]="heading1" [tabBody]="hello"></ez-tab>
          <ez-tab tabTitle="Extras">
              <div>Hello, User</div>
          </ez-tab>
          <ng-template #hello>
              <div>Extra Content here.</div>
          </ng-template> 
          <ng-template #heading1>
              <span class="fa fa-user"></span> <span style="padding-left: 10px">User</span>
          </ng-template>
        </ez-tabbar>
Properties: Tabbar
Name Values Default Description
styleClass string: custom NA Takes in given class name and applys to the tab container division.
headerStyleClass string: custom NA Takes in given class name and applys to the tab header container division.
bodyStyleClass string: custom NA Takes in given class name and applys to the tab body container division.
direction string: vertical / horizontal horizontal Defines the layout of the tabbar.
height number: (in pixels) NA Defines the fixed height of tabbar irrespective of content height.
scrollHt number: (in pixels) NA Defines the height of tabbar after which scroll will be added automatically.
Properties: Tab
Name Values Default Description
tabTitle string / templateRef NA Can provide a string or a ng-template ref.
tabBody string / templateRef NA Can provide a string or a ng-template ref.
active number: (index) NA Active tab to be selected on init.
Events: Tabbar
Name Parameter Description
tabSelect $event Returns the tabs list as an array.
