Hi, {{item?.label}}. How are you ?
Syntax Code Pattern 
        
        <ez-listbox [listData]="listData" [template]="listTmpl" styleClass="customStyleClass" selection="single" (onselection)="getSelectedItems($event)"></ez-listbox>
        <ng-template #listTmpl let-item="data">
            <div class="template_class">
                <i [ngClass]="item?.icon"></i>
                <span>
                  Hi, item?.label}}. How are you ?
                </span>
            </div>
        </ng-template>
Properties:
Name Values Mandatory / Default Description
listData JSON Object: Format: EzListBox Yes Receives the JOSN object for the list that needs to be displayed.
selection string: 'single' / 'multiple' 'single' For single selection or multiple selection.
template templateRef - Pass a ng-template reference that you want to be iterated inside each list item.
styleClass string - Custom class name to be added to the .ez-listbox element.
Events:
Name Parameter Description
onselection $event Returns the selected items (event.selectedData) list as well as original list (event.originalData).
