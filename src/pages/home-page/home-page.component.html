EzNgLib: An easy to use, flexible angular library

Developed keeping in mind, the issues, developers face, while integrating some 3rd party components / libraries. Some dont have the flexibility to add own template, while some dont have the option to have a callback method at every action of their component. These are some of the few issues addressed in this library.

While, this library is in its conception stage, as expected, there could be many issues alongwith it. Hence, as ever helping and kind contributors, its my request to you to keep suggesting changes and enchancements that would drive this concept forward. Also you can directly contribute in Github if you like to. ;)
